### Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- ReactJS
- Styled Components

### Como rodar o projeto

```bash
# Clone o repositório
git clone

# Entre no diretório
cd todo-list

# Instale as dependências
npm install

# Rode o projeto
npm start
```
