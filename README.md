### Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- ReactJS
- Styled Components

### Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/vbruno/FEMA-web3-atv02/

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
